# 🚨 AutoVid 복제 - 필수 수정사항

## ❌ 현재 문제점

**만든 프로그램**: 기본적인 데모 UI만 있음  
**원본 AutoVid**: 완전한 기능을 가진 프로덕션 앱

---

## ✅ 반드시 구현해야 할 12개 핵심 기능

### 1. 로그인 시스템
- 원본: Google 계정 로그인 + 프로필 표시
- 현재: 없음
- **→ Firebase Auth 또는 Google OAuth 2.0 구현 필수**

### 2. 크레딧 시스템
- 원본: S-CRD (무료), E-CRD (유료) 표시 및 관리
- 현재: 없음
- **→ 사용자별 크레딧 DB + 차감/충전 로직 구현**

### 3. 다운로드 기능
- 원본: YouTube URL 입력 → 비디오 다운로드
- 현재: 없음
- **→ yt-dlp 또는 youtube-dl 통합**

### 4. 수동영상생성
- 원본: 새 프로젝트 생성, 클립보드에서 이미지 붙여넣기, 타임라인 편집, 전환 효과 선택
- 현재: 기본 폼만 있음
- **→ 타임라인 에디터 + 이미지 업로드 + 미리보기 구현**

### 5. 자동영상생성 (AI)
- 원본: 프로젝트명, 출력옵션, 음성 설정, 템플릿 선택/수정, 자막 스타일, 배경음악 설정
- 현재: 일부 UI만 있음
- **→ 전체 워크플로우 구현 (스크립트 생성 → 이미지 생성 → 비디오 조립)**

### 6. 베스트시리즈
- 원본: 전체 자동 생성, 부분 자동 생성, 프로젝트 관리
- 현재: 없음
- **→ 시리즈 프로젝트 관리 기능**

### 7. Template 에디터
- 원본: 새 템플릿 생성, 텍스트/도형/이미지/스티커 추가, 상단/하단 높이 조절, 실시간 미리보기
- 현재: 없음
- **→ 완전한 템플릿 에디터 구현 필요**

### 8. 유튜브 탐색
- 원본: 내장 YouTube 브라우저
- 현재: 없음
- **→ WebView2 또는 iframe으로 YouTube 통합**

### 9. Fonts Download
- 원본: Google Fonts 검색 및 다운로드
- 현재: 없음
- **→ Google Fonts API 통합**

### 10. BGM
- 원본: Pixabay 음악 검색 및 다운로드
- 현재: 없음
- **→ Pixabay API 통합**

### 11. 프로필 설정
- 원본: 스크립트 언어, 프롬프트명, 자막 스타일, 채널 인트로/아우트로, YouTube API, Channel ID/Name
- 현재: 없음
- **→ 설정 관리 시스템 구현**

### 12. Shop (결제)
- 원본: 월간/연간 구독, 크레딧 충전, Microsoft Store 결제
- 현재: 없음
- **→ 결제 시스템 (Stripe 또는 PayPal) 통합**
